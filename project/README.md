

/*
MySQL 常用命令：

1. 连接到 MySQL：
   ```bash
   mysql -u root -p
   ```

2. 查找所有数据库：
   ```sql
   SHOW DATABASES;
   ```

3. 进入指定数据库：
   ```sql
   USE cloud_nav;
   ```

4. 查找数据库中的所有表：
   ```sql
   SHOW TABLES;
   ```

5. 查看表结构：
   ```sql
   DESCRIBE 表名;
   ```
   或：
   ```sql
   SHOW COLUMNS FROM 表名;
   ```

6. 查看表中的所有数据：
   ```sql
   SELECT * FROM 表名;
   ```

这些命令可帮助您快速管理 MySQL 数据库。
*/
